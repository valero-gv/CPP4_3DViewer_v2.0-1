Руководство по использованию 3D_Viewer_v2:

Установка программы:

1. Через терминал перейдите в каталог src.

2. Введите команду make.

Запуск программы:

1. Откройте каталог build.

2. Запустите приложение с помощью команды "./3DViewer"

Загрузка модели:

1. В главном окне нажмите кнопку "Open file" и выберите файл объекта – отобразится каркасная модель.

Управление моделью:

1. Перемещение: сдвигать модель можно с помощью ползунков в разделе "Translate Model" по осям x, y, z.

2. Поворот: вращать модель можно с помощью ползунков в разделе "Rotate Model" по осям x, y, z.

3. Масштабирование: изменять размеры модели можно с помощью ползунка в разделе "Scale Model" по осям x, y, z.

Настройки:

1. Проекция: можно изменять тип проекции через соответствующие кнопки.

2. Рёбра: можно настроить тип отображения рёбер, их толщину и цвет.

3. Вершины: можно выбрать тип отображения вершин, их размер и цвет рамки.

4. Фон: можно изменить цвет фона, нажав кнопку "Set Background".

5. Все настройки сохраняются между запусками программы.

Сохранение изображения:

1. Загрузите каркасную модель.

2. Нажмите кнопку "Screenshot", выберите формат изображения и папку для сохранения, затем нажмите "save" – скриншот модели будет сохранён.

Создание анимации (скринкаста):

1. Загрузите каркасную модель.

2. Чтобы создать GIF-анимацию текущих преобразований модели (размер 640x480, 10 кадров в секунду, 5 секунд), нажмите кнопку "GIF".

3. Укажите название файла, выберите папку для сохранения и нажмите "save".

4. Взаимодействуйте с моделью как вам нравится.

5. GIF-анимация будет сохранена в указанной папке.
